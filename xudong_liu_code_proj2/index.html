P1.1

I created a finite difference operator for the horizontal and vertical directions of the image. Dx = [1, -1] and Dy = [1, -1].T. Applying these filters will take the x partial derivative and y partial derivative, respectively, of the image. Then, I used these partial derivatives to calculate the gradient magnitude: Grad_Mag = SQRT(Dx**2 + Dy**2).

P1.3

I first converted all images to grayscale in order to convolve with the DoG filter. I iterated through a small range of angles (-4 degrees to 4 degrees in steps of 0.2 degrees). For each angle, after computing the DoG values for both the x-axis and y-axis, I computed the direction of each gradient using arctan(-y/x). A histogram is formed using the computed angle values.

P2.1

To sharpen an image, we can increase the high frequencies. By applying the gaussian filter as a low pass filter to an image, we can obtain the low frequencies of the image and then subtract the low frequencies from the original image to obtain the high frequencies.

A gaussian stack is created by starting with a gaussian kernel and at each level down the stack increasing the size and sigma of the kernel. A Laplacian stack is created by starting with a laplacian filter and at each level down the stack increasing the size and sigma of the kernel.
